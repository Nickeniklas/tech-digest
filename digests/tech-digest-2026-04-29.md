## Under the Hood

### Quarkdown: Scripted Markdown That Compiles to Books, Websites, and Slides

**What happened**
Quarkdown is a new open-source Markdown dialect that adds functions, variables, conditionals, and layout directives. A single source file compiles to HTML, PDF, or presentation formats — no separate build tooling required.

**Why it matters**
Most document tools lock you into one output format. Quarkdown treats your document as source code: write logic and content together, then publish to any format. It ships with a standard library for academic and technical writing, making it a practical option for technical writers and researchers who want one source of truth.

Source: [GitHub Trending ↗](https://github.com/iamgio/quarkdown)
