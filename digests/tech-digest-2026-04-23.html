claude-context: Semantic Whole-Repo Search as an MCP for Claude Code

claude-context is a new MCP server from Zilliz that gives Claude Code agents semantic code search over your entire codebase — no manual file referencing or context stuffing required.

Why it matters

Context window management is the #1 friction point in AI-assisted coding today. An MCP that handles retrieval automatically means your agent can navigate large repos without you babysitting which files to include in every prompt.

TRY IT
# Add to your Claude Code MCP config (claude_desktop_config.json or .claude/settings.json)
{
  "mcpServers": {
    "claude-context": {
      "command": "npx",
      "args": ["-y", "@zilliz/claude-context"]
    }
  }
}

Langfuse Adds OpenTelemetry Integration for LLM Observability

Langfuse is trending as one of the top open-source LLM engineering platforms, offering observability, evals, prompt management, and datasets — now with OpenTelemetry support for standardized tracing across your stack.

TRY IT
pip install langfuse

trace = langfuse.trace(name="my-llm-call")
span = trace.span(name="generation")
# wrap your LLM call here

Shannon Lite: Autonomous White-Box AI Pentester for APIs

Shannon Lite is an autonomous AI security tool that reads your source code directly, maps attack vectors, and generates exploit chains to test your own web applications and APIs.

THE TAKE

White-box analysis is the right approach here — black-box fuzzing misses too much. The real question is whether teams use this as a continuous gate in CI or a one-off audit tool. The former is where this gets genuinely useful.
